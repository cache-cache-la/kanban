# Overview
Python, Django学習のために作成した、Trello風アプリケーションです。

# Description
以下の機能を実装しています。
* ユーザー新規登録/ログイン
* ユーザーマイページ
  * ユーザー名、姓名、メールアドレス編集／更新
* リスト／カード機能
  * 新規投稿、詳細表示、編集、削除
* ページ共通
  * ヘッダー固定
